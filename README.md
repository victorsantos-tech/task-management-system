
# Task Management System

Aplicação Full Stack para gerenciamento de tarefas.

## 🚀 Tecnologias
- React
- Flask
- Python
- JavaScript

## ✅ Funcionalidades
- Criar tarefas
- Listar tarefas
- Deletar tarefas
- Integração entre frontend e backend

## ▶️ Como rodar

### Backend:
cd backend
python app.py

### Frontend:
cd frontend
npm start
